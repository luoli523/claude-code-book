# 前言

## 为什么写这本书

2024 年底，Anthropic 发布了 Claude Code。和其他代码补全工具不同，Claude Code 是真正的"代理式"编程助手——它不只是帮你补全代码，而是真正理解你的整个项目，规划方案，跨文件执行，自己验证结果。

我在学习官方课程和文档的过程中，发现市面上缺少一本把"是什么、为什么、怎么用"都讲清楚的系统性参考书。所以就有了这本书。

## 本书的目标

- **不只是功能罗列**：每个特性都会解释背后的设计思路
- **配合实际场景**：大量真实项目场景中的使用案例
- **踩坑经验**：那些官方文档没写、但你一定会遇到的问题
- **可以动手**：每章都有可以直接运行的示例

## 内容来源

本书内容来自：
1. Anthropic 官方课程《Claude Code in Action》
2. Claude Code 官方文档（code.claude.com/docs）
3. 实际使用中的经验积累
4. 社区分享的最佳实践

## 读者要求

- 会用命令行（不需要专家级别）
- 了解 Git 基础操作
- 有一定编程经验（任何语言均可）

## 约定说明

本书中的代码示例使用以下约定：

```bash
# 以 $ 开头的是终端命令
$ claude "帮我分析这个项目"

# 以 > 开头的是 Claude Code 交互提示
> what does this project do?
```

配置文件示例会标注文件路径：

```json
// ~/.claude/settings.json
{
  "hooks": { ... }
}
```
