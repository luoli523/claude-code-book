# 跟鬼哥玩转 Claude Code

📖 **在线阅读**：[luoli523.github.io/claude-code-book](https://luoli523.github.io/claude-code-book/)

## 关于本书

本书系统整理了 Claude Code 的核心概念、使用技巧与实战经验，内容来自：

- Anthropic 官方课程《Claude Code in Action》
- Claude Code 官方文档
- 实际使用中的踩坑经验与最佳实践

适合想把 AI 融入日常开发工作流的软件开发者、希望在团队落地 AI 编程助手的技术 leader，以及对 AI Agent 架构感兴趣的技术研究者。
