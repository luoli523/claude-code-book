# 跟鬼哥一起玩转 Claude Code

> 从零到精通 Anthropic Claude Code — 一本面向开发者的完整参考书

---

## 关于本书

本书系统整理了 Claude Code 的核心概念、使用技巧与实战经验，适合：

- 想把 AI 融入日常开发工作流的**软件开发者**
- 希望在团队中落地 AI 编程助手的**技术 leader**
- 对 AI Agent 架构感兴趣的**技术研究者**

本书基于 Anthropic 官方课程《Claude Code in Action》及官方文档编写，并加入大量实战案例与踩坑经验。
