# 前言

> 从零到精通 Anthropic Claude Code — 一本面向开发者的完整参考书

![前言](/images/preface.webp)

## 为什么写这本书

2024 年底，Anthropic 发布了 Claude Code。和其他代码补全工具不同，Claude Code 是真正的"代理式"编程助手——它不只是帮你补全代码，而是真正理解你的整个项目，规划方案，跨文件执行，自己验证结果。

我在学习和使用Claude Code的过程中，是一边实际用它来做开发, 一边摸索它的各种用法和最佳实践. 市面上有不少使用者的使用经验和分享, 官方的文档也很多很系统, 我自己着实啃了不少原文文档, 做了不好凌乱的笔记.

最近发现 Anthropic 也把Claude Code的帮助文档整理成了一门课程, 也就是 [《Claude Code in Action》](https://anthropic.skilljar.com/claude-code-in-action) 这门课程, 我也花了点时间读了一遍, 发现很多内容跟我的笔记都契合, 所以就整理了一遍自己凌乱的笔记, 结合阅读官方课程的理解, 整理成了这本书。

## 本书的目标

- **不只是功能罗列**：每个特性都会解释背后的设计思路
- **配合实际场景**：大量真实项目场景中的使用案例
- **踩坑经验**：那些官方文档没写、但你一定会遇到的问题
- **可以动手**：每章都有可以直接运行的示例

## 内容来源

本书内容来自：
1. Anthropic 官方课程《Claude Code in Action》
2. Claude Code 官方文档（code.claude.com/docs）
3. 实际使用中的经验积累
4. 社区分享的最佳实践

## 约定说明

本书中的代码示例使用以下约定：

```bash
# 以 $ 开头的是终端命令
$ claude "帮我分析这个项目"

# 以 > 开头的是 Claude Code 交互提示
> what does this project do?
```

配置文件示例会标注文件路径：

```json
// ~/.claude/settings.json
{
  "hooks": { ... }
}
```
