# 第六章：MCP 服务器集成

到目前为止，Claude Code 一直在你的电脑上工作——读你的文件，跑你的命令，改你的代码。

但真实的开发工作不只在本地发生。你有 GitHub 上的 issue 要处理，有 Slack 上的通知要发，有数据库要查询，有设计稿要参考……

**MCP 就是让 Claude Code 伸出手，触达这些外部世界的方式。**
