# 第一章：Claude Code 是什么？

如果你曾经盯着一个 bug 看了两个小时，最后发现是漏了一个分号——欢迎来到编程的世界。

Claude Code 不能帮你避免所有这样的时刻，但它可以让你少经历很多。

本章不讲安装，不讲命令，只做一件事：**让你真正理解 Claude Code 是什么，以及它为什么和你用过的其他工具不一样**。
