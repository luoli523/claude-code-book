# 第三章：上下文管理

有一件事很多人用了一段时间 Claude Code 之后才意识到：

**它用着用着，感觉变蠢了。**

给出的答案越来越模糊，开始忘记你之前说过的要求，改出的代码开始走偏……

这不是 bug，这是上下文窗口填满了的正常现象。

本章就是专门解决这个问题的。上下文管理是用好 Claude Code 的核心技能，但也是最容易被忽视的一个。把这一章学透，你会感觉像是换了一个更聪明的 AI。
