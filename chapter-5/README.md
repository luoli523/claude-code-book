# 第五章：自定义命令与自动化

你有没有发现，有些任务你每次都要重新描述一遍：

- "按照我们团队的规范帮我 review 这个 PR，检查安全性、性能、测试覆盖……"
- "帮我生成这次改动的 changelog，格式要符合 Keep a Changelog 标准……"
- "部署到 staging 之前先跑完整测试，检查环境变量，然后……"

说一次两次还好，说第十次你就会开始想：**能不能把这些打包成一个命令？**

可以，这就是 Skills（自定义命令）。
